# Skeleton 骨架屏

## 介绍

在需要等待加载内容的位置提供一个占位图形组合

## 使用指南

文件中引入组件

```jsx
import React from 'react'
import { HuiSkeleton } from '@zaihui/hui-design'

const Demo: React.FC = () => <Skeleton />

export default Demo
```
